# 📊 預測行為診斷與效能分析系統 (Predictive Behavior & Forecast Analysis System)

這是一個專為製造業與供應鏈營運設計的 **全方位產銷績效決策看板**。
透過整合 **生管實務流程**、**財會成本嚴謹邏輯** 與 **IT 數據自動化技術**，協助企業快速診斷前端業務 Forecast 的準確度與變動行為。

## 💡 核心功能與管理價值
* **FVA (Forecast Value Add)  white-box 拆解**：量化人工介入修改對比系統初版的實質提升價值。
* **樂觀偏誤 (Optimism Bias) 診斷**：自動識別業務是否存在「防禦性寬估（多報）」或「嚴重低估（少報）」的行為傾向，提早預警呆滯與缺貨風險。
* **多維度動態鑽取 (Drill-down)**：提供管理層、供應鏈、PP (生產計劃) 三大視角切換，點擊圖表即可動態解鎖單一品項的「預測演變軌跡」與「客觀 AI 診斷建議」。

## 🛠 使用技術
* **Frontend / Web App**: Streamlit
* **Data Processing**: Python (Pandas, NumPy)
* **Data Visualization**: Plotly (Dynamic Scatter, Go-Bar, Interactive Plots)
* **Analytics**: Scikit-Learn (Linear Regression), ACCBE 經營分析邏輯
